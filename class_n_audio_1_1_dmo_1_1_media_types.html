static string NAudio.Dmo.MediaTypes.GetMediaTypeName ( Guid majorType)
inlinestatic

Returns the media type name for the given major type.

Parameters
majorTypeThe major type for which the media type name is to be retrieved.
Returns
The media type name corresponding to the specified majorType .
Exceptions
ArgumentExceptionThrown when the specified majorType is not found in the MajorTypes array.

This method iterates through the MajorTypes array to find a match for the given majorType . If a match is found, the corresponding media type name is returned. If no match is found, an ArgumentException is thrown with the message "Major Type not found".
