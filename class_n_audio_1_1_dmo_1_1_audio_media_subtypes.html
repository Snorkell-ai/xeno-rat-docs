static string NAudio.Dmo.AudioMediaSubtypes.GetAudioSubtypeName ( Guid subType)
inlinestatic

Gets the name of the audio sub type based on the provided GUID.

Parameters
subTypeThe GUID of the audio sub type.
Returns
The name of the audio sub type corresponding to the provided subType . If no matching sub type is found, the method returns the string representation of the input subType .
