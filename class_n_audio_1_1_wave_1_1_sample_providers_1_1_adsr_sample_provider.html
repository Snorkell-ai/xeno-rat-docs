int offset,
int count )
inline

Reads audio samples from the source, processes them using the ADSR envelope generator, and returns the number of samples read.

Parameters
bufferThe buffer to store the audio samples.
offsetThe zero-based offset in the buffer at which to begin storing the data.
countThe maximum number of samples to read.
Returns
The number of samples read from the source after processing with the ADSR envelope generator.

If the ADSR envelope generator is in the idle state, indicating that the audio has finished, this method returns 0. This method reads audio samples from the source into the buffer, processes each sample using the ADSR envelope generator, and then returns the total number of samples read.
