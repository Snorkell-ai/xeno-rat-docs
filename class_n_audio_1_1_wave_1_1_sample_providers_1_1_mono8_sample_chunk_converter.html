out float sampleRight )
inline

Retrieves the next sample from the source buffer and returns it as the left and right audio samples.

Parameters
sampleLeftThe variable to store the left audio sample.
sampleRightThe variable to store the right audio sample.
Returns
True if the next sample is successfully retrieved; otherwise, false.

This method retrieves the next sample from the source buffer and assigns it to the sampleLeft and sampleRight parameters. If the end of the source buffer is reached, the method returns false and assigns 0.0f to both sampleLeft and sampleRight .

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.

int samplePairsRequired )
inline

Loads the next chunk of audio data from the provided wave source.

Parameters
sourceThe wave provider from which to load the audio data.
samplePairsRequiredThe number of sample pairs required for the next chunk.

This method ensures that the source buffer is large enough to accommodate the required number of sample pairs. It then reads the audio data from the source into the source buffer and resets the offset to 0 for subsequent processing.
