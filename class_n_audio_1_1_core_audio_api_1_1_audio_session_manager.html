void NAudio.CoreAudioApi.AudioSessionManager.RefreshSessions ( )
inline

Refreshes the audio sessions and registers for notifications.

This method unregisters any existing notifications, retrieves the session enumerator, and creates a new session collection. It then registers for session notifications and throws an exception if any of the underlying COM operations fail.

◆ UnregisterNotifications()

void NAudio.CoreAudioApi.AudioSessionManager.UnregisterNotifications ( )
inlineprivate

Unregisters the notifications for audio sessions and sets the sessions to null.

This method sets the sessions to null and unregisters the audio session notification if audioSessionNotification and audioSessionInterface2 are not null. If audioSessionNotification and audioSessionInterface2 are not null, it calls the Marshal.ThrowExceptionForHR(int) method with the HRESULT returned by IAudioSessionControl2.UnregisterSessionNotification(IAudioSessionEvents).
