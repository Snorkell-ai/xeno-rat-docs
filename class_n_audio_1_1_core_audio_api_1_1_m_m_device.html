void NAudio.CoreAudioApi.MMDevice.Dispose ( )
inline

Performs application-defined tasks associated with freeing, releasing, or resetting unmanaged resources.

This method disposes of the audioEndpointVolume and audioSessionManager objects if they are not null. It also suppresses the finalization of the current object to prevent the finalizer from being called.
