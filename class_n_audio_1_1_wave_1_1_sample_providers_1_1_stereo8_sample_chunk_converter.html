Retrieves the next pair of samples from the source buffer and returns them as out parameters.

Parameters
sampleLeftThe left channel sample retrieved from the source buffer.
sampleRightThe right channel sample retrieved from the source buffer.
Returns
True if there are more samples available in the source buffer; otherwise, false.

This method retrieves the next pair of samples from the source buffer and returns them as out parameters. If there are more samples available in the source buffer, it returns true and updates the out parameters with the samples. If there are no more samples available, it returns false and updates the out parameters with 0.0f.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.

Loads the next chunk of audio data from the specified wave provider.

Parameters
sourceThe wave provider from which to load the audio data.
samplePairsRequiredThe number of sample pairs required for the next chunk of audio data.

This method loads the next chunk of audio data from the specified wave provider. It calculates the number of bytes required based on the sample pairs required and ensures that the source buffer is large enough to hold the data. The method then reads the specified number of bytes from the wave provider into the source buffer and resets the offset to 0.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.
