◆ Dispose()

void NAudio.CoreAudioApi.AudioEndpointVolume.Dispose ( )
inline

Performs application-defined tasks associated with freeing, releasing, or resetting unmanaged resources.

This method releases the unmanaged resources used by the audioEndPointVolume object and optionally releases the managed resources. If the callBack is not null, it unregisters the control change notification for the audioEndPointVolume. It then releases the COM object and suppresses the finalization of this object.

Exceptions
System.Runtime.InteropServices.COMExceptionThrown when an error is encountered while unregistering the control change notification for the audioEndPointVolume.
