bool NAudio.Wave.SampleProviders.Mono16SampleChunkConverter.GetNextSample ( out float sampleLeft,
out float sampleRight )
inline

Retrieves the next sample from the source wave buffer and returns it as floating-point values for the left and right channels.

Parameters
sampleLeftWhen this method returns, contains the next sample value for the left channel.
sampleRightWhen this method returns, contains the next sample value for the right channel.
Returns
True if the next sample was successfully retrieved; otherwise, false.

If the source sample index is less than the total number of source samples, this method retrieves the next sample from the source wave buffer and returns it as floating-point values for both the left and right channels. If the source sample index is equal to or greater than the total number of source samples, this method returns 0.0f for both the left and right channels and false to indicate that no more samples are available.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.

Loads the next chunk of audio data from the specified wave provider.

Parameters
sourceThe wave provider from which to load the audio data.
samplePairsRequiredThe number of sample pairs required for the next chunk.

This method calculates the number of bytes required from the number of sample pairs and initializes the necessary buffers. It then reads the audio data from the source wave provider into the source buffer and updates the source sample and source samples fields accordingly.
