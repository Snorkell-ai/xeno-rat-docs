void NAudio.FileFormats.Wav.WaveFileChunkReader.ReadWaveHeader ( Stream stream)
inline

Reads the wave header from the provided stream and initializes the necessary fields.

Parameters
streamThe input stream containing the wave file data.
Exceptions
FormatExceptionThrown when the input stream does not contain a WAVE header.
InvalidDataExceptionThrown when the format chunk length or riff chunk length is invalid.

This method reads the wave header from the provided stream and initializes the necessary fields such as dataChunkPosition, waveFormat, riffChunks, and dataChunkLength. It first reads the RIFF header, then the file size, and checks for the WAVE header. If the file is in RF64 format, it reads the DS64 chunk. It then iterates through the chunks in the stream, processing the data and format chunks accordingly. The method also handles word alignment for all chunks and throws exceptions if the format or data chunk is not found in the input stream.
