int offset,
int count )
inline

Reads data from the source into the buffer and returns the number of frames read.

Parameters
bufferThe buffer to store the read data.
offsetThe zero-based byte offset in buffer at which to begin storing the data read from the current stream.
countThe maximum number of frames to read.
Exceptions
System.ArgumentNullExceptionThrown when the buffer is null.
System.ArgumentOutOfRangeExceptionThrown when offset or count is less than 0.
Returns
The number of frames read into the buffer.

This method reads data from the source into the buffer and performs resampling if necessary to match the requested number of frames. It returns the actual number of frames read into the buffer after resampling.
