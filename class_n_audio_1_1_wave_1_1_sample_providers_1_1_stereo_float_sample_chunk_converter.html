out float sampleRight )
inline

Retrieves the next pair of samples from the wave buffer and returns them as out parameters.

Parameters
sampleLeftThe variable to store the left sample.
sampleRightThe variable to store the right sample.
Returns
True if the next samples are successfully retrieved; otherwise, false.

This method retrieves the next pair of samples from the wave buffer and stores them in the out parameters sampleLeft and sampleRight . If there are no more samples to retrieve, the method returns false and sets both sampleLeft and sampleRight to 0.0f.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.

void NAudio.Wave.SampleProviders.StereoFloatSampleChunkConverter.LoadNextChunk ( IWaveProvider source,
int samplePairsRequired )
inline

Loads the next chunk of audio data from the specified source and prepares it for processing.

Parameters
sourceThe audio source from which the data will be loaded.
samplePairsRequiredThe number of sample pairs required for processing the next chunk.

This method calculates the number of bytes required from the samplePairsRequired and ensures that the sourceBuffer is of sufficient size to accommodate the data. It then reads the required data from the source into the sourceBuffer and initializes the sourceWaveBuffer for processing. The number of samples read from the source is calculated and stored in sourceSamples , and the sourceSample index is reset to 0 for processing.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.
