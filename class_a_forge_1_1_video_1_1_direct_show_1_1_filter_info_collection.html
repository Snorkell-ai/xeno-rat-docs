Detailed Description

Collection of filters' information objects.

The class allows to enumerate DirectShow filters of specified category. For a list of categories see FilterCategory.

Sample usage:

// enumerate video devices
videoDevices = new FilterInfoCollection( FilterCategory.VideoInputDevice );
// list devices
foreach ( FilterInfo device in videoDevices )
{
// ...
}

Collects filters of a specified category and adds them to the collection.

Parameters
categoryThe category of filters to be collected.
Exceptions
ApplicationExceptionThrown when there is a failure in creating the device enumerator or no devices of the category are found.

This method collects filters of the specified category using the system device enumerator and adds them to the collection. It iterates through the filters, creates FilterInfo objects, and adds them to the collection. The collection is then sorted.
