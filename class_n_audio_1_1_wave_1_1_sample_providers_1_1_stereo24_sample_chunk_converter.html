out float sampleRight )
inline

Retrieves the next audio sample from the source buffer and returns it as two floating-point values.

Parameters
sampleLeftThe variable to store the left channel audio sample.
sampleRightThe variable to store the right channel audio sample.
Returns
True if the next sample is successfully retrieved; otherwise, false.

This method retrieves the next audio sample from the source buffer and stores it in the variables sampleLeft and sampleRight . The method also advances the internal offset to the next set of samples in the source buffer. If the end of the source buffer is reached, the method returns false and sets both sampleLeft and sampleRight to 0.0f.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.

Loads the next chunk of audio data from the specified wave provider.

Parameters
sourceThe wave provider from which to load the audio data.
samplePairsRequiredThe number of sample pairs required for the next chunk.
Exceptions
ArgumentNullExceptionThrown when the source is null.
InvalidOperationExceptionThrown when the source does not contain enough data to fulfill the required sample pairs.

This method loads the next chunk of audio data from the specified wave provider. It calculates the number of bytes required based on the sample pairs required and ensures that the source buffer is large enough to hold the data. It then reads the specified number of bytes from the wave provider into the source buffer and sets the offset to 0.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.
