Creates an all-pass filter using the specified sample rate, center frequency, and Q value.

Parameters
sampleRateThe sample rate of the audio signal.
centreFrequencyThe center frequency of the filter.
qThe Q value of the filter.
Returns
A BiQuadFilter representing the all-pass filter.

This method calculates the coefficients for an all-pass filter based on the provided sample rate, center frequency, and Q value. The all-pass filter is defined by the transfer function H(s) = (s^2 - s/Q + 1) / (s^2 + s/Q + 1), where s is the Laplace transform variable. The method computes the coefficients b0, b1, b2, a0, a1, and a2 using the provided parameters and returns a new BiQuadFilter instance initialized with these coefficients.

static BiQuadFilter NAudio.Dsp.BiQuadFilter.HighShelf ( float sampleRate,

float shelfSlope,

Calculates the coefficients for a high-shelf filter using the given parameters and returns a BiQuadFilter object.

Parameters
sampleRateThe sample rate of the audio signal.
cutoffFrequencyThe cutoff frequency of the filter.
shelfSlopeThe slope of the shelf.
dbGainThe gain in decibels for the shelf filter.
Returns
A BiQuadFilter object with the calculated coefficients for the high-shelf filter.

This method calculates the coefficients for a high-shelf filter using the given parameters based on the following formulas:

  • w0 = 2 * Ï€ * cutoffFrequency / sampleRate
  • cosw0 = Cos(w0)
  • sinw0 = Sin(w0)
  • a = 10^(dbGain / 40)
  • alpha = sinw0 / 2 * Sqrt((a + 1 / a) * (1 / shelfSlope - 1) + 2)
  • temp = 2 * Sqrt(a) * alpha
  • b0 = a * ((a + 1) + (a - 1) * cosw0 + temp)
  • b1 = -2 * a * ((a - 1) + (a + 1) * cosw0)
  • b2 = a * ((a + 1) + (a - 1) * cosw0 - temp)
  • a0 = (a + 1) - (a - 1) * cosw0 + temp
  • a1 = 2 * ((a - 1) - (a + 1) * cosw0)
  • a2 = (a + 1) - (a - 1) * cosw0 - temp The calculated coefficients are used to create a BiQuadFilter object, which is then returned.

This method calculates the coefficients for a low-pass filter based on the given sample rate, cutoff frequency, and Q value using the following transfer function: H(s) = 1 / (s^2 + s/Q + 1) where s = jw (j is the imaginary unit and w is the frequency in radians per second). The coefficients are then set using the SetCoefficients method to be used in the filter implementation.
