Reads audio samples from the source provider and writes them to the destination buffer.

Parameters
destBufferThe destination buffer to write the audio samples to.
offsetThe offset in the destination buffer at which to start writing.
numBytesThe number of bytes to read from the source provider and write to the destination buffer.
Returns
The actual number of bytes written to the destination buffer.

This method reads audio samples from the source provider, adjusts their volume, clips them if necessary, converts them to 24-bit samples, and writes them to the destination buffer. If the number of bytes requested is not a multiple of 3, the remaining bytes in the destination buffer will not be written to.
