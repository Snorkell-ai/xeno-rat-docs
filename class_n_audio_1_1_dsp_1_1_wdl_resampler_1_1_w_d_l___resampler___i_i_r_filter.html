Applies a filter to the input buffer and stores the result in the output buffer.

Parameters
inBufferThe input buffer containing samples to be filtered.
inIndexThe index in the input buffer to start filtering from.
outBufferThe output buffer to store the filtered samples.
outIndexThe index in the output buffer to start storing the filtered samples.
nsThe number of samples to filter.
spanThe span between samples in the input and output buffers.
wThe value used for indexing in the history buffer.

This method applies a filter to the input buffer using the coefficients b0, b1, b2, a1, and a2. The filtered samples are stored in the output buffer starting from the specified index. The history buffer m_hist is used to store previous input and output samples for filtering.

double NAudio.Dsp.WdlResampler.WDL_Resampler_IIRFilter.denormal_filter ( float x)
inlineprivate

Filters the input value to prevent denormalization and returns the filtered value.

Parameters
xThe input value to be filtered.
Returns
The filtered value of x .

This method is intended to prevent denormalization of the input value x and returns the filtered value. Denormalization occurs when a floating-point number is too close to zero, causing a loss of precision and performance degradation. The implementation of denormalization filtering is pending and needs to be completed.
