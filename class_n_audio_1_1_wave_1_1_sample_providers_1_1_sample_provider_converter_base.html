void NAudio.Wave.SampleProviders.SampleProviderConverterBase.EnsureSourceBuffer ( int sourceBytesRequired)
inlineprotected

Ensures that the source buffer has enough capacity to accommodate the specified number of bytes required.

Parameters
sourceBytesRequiredThe number of bytes required for the source buffer.

This method ensures that the source buffer has enough capacity to accommodate the specified number of bytes required. If the current capacity of the source buffer is less than the specified number of bytes required, it reallocates the buffer with the new capacity. The source buffer is modified in place to ensure sufficient capacity.
