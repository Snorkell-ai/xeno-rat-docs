bool NAudio.Wave.SampleProviders.Stereo16SampleChunkConverter.GetNextSample ( out float sampleLeft,
out float sampleRight )
inline

Retrieves the next pair of audio samples from the source buffer and returns them as floating-point values.

Parameters
sampleLeftWhen this method returns, contains the next audio sample from the left channel as a floating-point value.
sampleRightWhen this method returns, contains the next audio sample from the right channel as a floating-point value.
Returns
True if the next samples are successfully retrieved; otherwise, false.

This method retrieves the next pair of audio samples from the source buffer and converts them to floating-point values by dividing the sample values by 32768.0f. If there are no more samples available in the source buffer, this method returns false and sets both sampleLeft and sampleRight to 0.0f.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.

int samplePairsRequired )
inline

Loads the next chunk of audio data from the specified wave provider.

Parameters
sourceThe wave provider from which to load the audio data.
samplePairsRequiredThe number of sample pairs required for the next chunk.

This method calculates the number of bytes required from the wave provider based on the specified number of sample pairs required. It then ensures that the source buffer is large enough to accommodate the required number of bytes. The method reads the audio data into the source buffer, converts it to sample pairs, and updates the source sample and source samples fields accordingly.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.
