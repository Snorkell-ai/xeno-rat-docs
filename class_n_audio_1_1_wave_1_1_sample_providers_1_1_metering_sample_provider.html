Detailed Description

Simple SampleProvider that passes through audio unchanged and raises an event every n samples with the maximum sample value from the period for metering purposes.

Reads audio samples from the source into the buffer and updates the stream volume.

Parameters
bufferThe buffer to store the audio samples.
offsetThe zero-based byte offset in the buffer at which to begin storing the data.
countThe maximum number of samples to read.
Returns
The total number of samples read into the buffer.

This method reads audio samples from the source into the buffer, updates the stream volume, and returns the total number of samples read. If there is an event listener for stream volume, it updates the volume and triggers the event when the sample count reaches the specified threshold (SamplesPerNotification).
