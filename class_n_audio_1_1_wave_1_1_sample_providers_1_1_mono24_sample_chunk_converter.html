out float sampleRight )
inline

Retrieves the next audio sample from the source buffer.

Parameters
sampleLeftThe left channel audio sample.
sampleRightThe right channel audio sample.
Returns
True if the next sample was successfully retrieved; otherwise, false.

This method retrieves the next audio sample from the source buffer. It interprets the bytes in the source buffer as 24-bit signed PCM audio samples and converts them to floating-point values in the range [-1.0, 1.0]. If there are remaining bytes in the source buffer, this method retrieves the next sample and advances the offset. If there are no remaining bytes, it returns false and sets both sampleLeft and sampleRight to 0.0f.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.

int samplePairsRequired )
inline

Loads the next chunk of audio data from the specified source and prepares it for processing.

Parameters
sourceThe wave provider from which the audio data will be loaded.
samplePairsRequiredThe number of sample pairs required for processing the next chunk of audio data.

This method calculates the number of bytes required from the source based on the samplePairsRequired . It then ensures that the sourceBuffer is of sufficient size to accommodate the required bytes using the BufferHelpers.Ensure method. The method reads the required bytes from the source into the sourceBuffer and sets the sourceBytes to the number of bytes read. The offset is reset to 0, indicating that the next chunk of audio data is ready for processing.

Implements NAudio.Wave.SampleProviders.ISampleChunkConverter.
